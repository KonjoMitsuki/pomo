# 学んだこと
## pythonには型ヒントがあるらしい。
`def 関数名(引数: 型名) -> 型名`で型ヒントの構文になる。
`from __future__ import annotations`で型ヒントが全部文字列として遅延評価される。
以下の例の場合、annotionsなしだとエラーになるがありだと素通りになりセーフらしい。 
```py
def create() -> "PomoSession": 
    pass

class PomoSession:
        pass
```
    
## dataclassについて
classに@dataclassとデコレータをつけるだけでコンストラクタを自動生成してくれる。
> デコレータとは
> classの拡張機能みたいなもので、変数や関数に機能を追加する仕組みである。
```py
@dataclass を使うと、フィールドを宣言するだけでコンストラクタが自動で作られます。
python@dataclass
class PomoSession:
    host_id: int        # フィールドの宣言だけ書く
    work_min: int = 25  # デフォルト値も設定できる

# 自動でこういうコンストラクタが生成される
# def __init__(self, host_id: int, work_min: int = 25):
#     self.host_id = host_id
#     self.work_min = work_min

# 使うときはこう
session = PomoSession(host_id=123456, work_min=25)
print(session.host_id)  # → 123456
```
